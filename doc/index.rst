Installation
------------

.. pypi-release:: xtraceback
      :prefix: The package is on PyPI

To install using :pypi:`pip`::

    pip install xtraceback

To install with syntax highlighting provided by :pypi:`Pygments`::

    pip install "xtraceback[syntax]"
